LETIMER_One_Shot_Output_Compare

This project demonstrates output compare in EM2 using the LETIMER.
The project initializes the timer in one-shot mode, and after one 
second, while staying in a low energy mode, will set either an Exp 
Header Pin or LED0 high.

How To Test:
1. Build the project and download to the Starter Kit
2. Measure output on the appropriate pin listed below or observe
LED0 light up

Peripherals Used:
LETIMER0 - one-shot mode
